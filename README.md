# AlgorithmCampus Frontend

GitHub 커밋 히스토리를 기반으로 알고리즘 풀이 패턴을 분석하고, 약점을 파악해 맞춤형 문제를 추천하는 플랫폼입니다.

## 주요 기능

- **내 분석** — 총 해결 문제 수, 현재 티어, 연속 풀이 일수, 난이도별/유형별 분포 시각화
- **추천 문제** — 풀이 패턴 분석 기반 맞춤형 문제 추천
- **풀이 기록** — 문제별 풀이 이력 및 필터링
- **북마크** — 나중에 풀 문제 저장
- **GitHub 연동** — 커밋 히스토리 자동 분석 및 동기화

## 기술 스택

- **React 19**
- **React Router v7**
- **Tailwind CSS v3**
- **Axios**
